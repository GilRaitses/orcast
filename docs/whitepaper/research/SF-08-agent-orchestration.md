## Family summary (updated — round 2)

The second round closed the primary gap. **ReAct (2210.03629)** is the canonical plan-act-verify paper: it demonstrates that interleaving reasoning traces with deterministic action execution substantially reduces hallucination versus chain-of-thought alone, and explicitly shows that constraining the action space to a defined tool set is what makes the verify step tractable. **Internal Representations as Indicators of Hallucinations in Agent Tool Selection (2601.05214)** directly measures hallucination rates in tool calling — finding that LLMs produce incorrect tool choices and malformed parameters at measurable rates, and that internal representations can predict these before execution — which is the empirical basis for the claim that manifest-constrained dispatch (catching invalid tools before they run) reduces error. **APIGen (2406.18518)** introduces the concept of verifiable function-calling datasets: it pipelines automated verification of whether a function call is valid before it enters training, establishing that tool-call verifiability is a tractable engineering property. The LLM Compiler for Parallel Function Calling (2312.04511) demonstrates that structured, compiler-validated function dispatch enables parallelism and reduces errors that arise from freeform sequential tool selection. The HaluAgent paper (2406.11277, round 1) remains the closest to the allow-list pattern: a structured toolbox with defined tool types enables smaller models to reliably detect hallucinations across categories. Taken together, these papers establish that plan-then-execute with manifest-constrained tool dispatch reduces hallucination and improves verifiability — exactly the pattern orcast's `skills_manifest.json` + `tier_blocked` 400 enforcement implements.

**Verdict: Supported** (upgraded from Partial). Primary citations: 2210.03629 (ReAct plan-act-verify), 2601.05214 (tool selection hallucination detection via internal representations), 2406.18518 (APIGen verifiable function calling), 2406.11277 (structured toolbox reduces hallucination categories).
