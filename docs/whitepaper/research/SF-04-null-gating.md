## Family summary

The phase-shuffled null test query did not return papers specific to cyclic ecological covariate testing — the top results addressed cyclostationary radio signals, quantum phase geometry, and video contrastive learning, indicating this specific methodology is not well-indexed on arXiv as used in neuroscience and ecology. The time-rescaling KS goodness-of-fit query returned strong goodness-of-fit literature (kernelized Stein discrepancy, KS statistics) as well as the point process theory literature (limit theory, 1606.03988) that formally underpins the time-rescaling theorem. The spike-train temporal model query returned directly relevant neuroscience papers (1506.02361, 2010.04875) that use point process models with spike trains — the same mathematical substrate that orcast's PSTH kernel adapts to encounter events. The permutation null distribution query returned valid statistical literature (permutation tests under rotating plans, 2004.13892) that establishes permutation/phase-shuffle methods as valid nonparametric null distributions. The model selection confidence bound query returned useful time-series forecasting papers (model confidence bounds, 1611.09509). The overall picture is that the statistical tools exist and are well-supported in the literature, but the specific application of phase-shuffled nulls to diel/lunar cyclic covariates in ecological point processes is documented in specialized ecology journals rather than arXiv CS, which is an honest corpus gap.

**Verdict: Partial** — the mathematical foundations of phase-shuffle nulls and time-rescaling GoF are well-established; their specific application to cyclic ecological covariates lacks an arXiv citation. Cite 2010.04875 (point process sequence detection) and 1506.02361 (point process spike train models) as methodological precedents, and note the ecology-journal gap.
